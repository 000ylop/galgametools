How to use Grapholic:
Archieve File:
1)You must know the file type of the archieve based on the 
  japanese company name found in this list.
2)Right click on the archieve & select Change Graphic Type---T.
3)Double click on the archieve.
4)The archieve is decompressed into a temp folder & list all files.
5)See Graphic File to convert those files into BMP.

Graphic File:
1)You must know the file type of the file based on the 
  japanese company name found in this list.
2)Right click on the file & Change Graphic Type---T.
3)Right click on the file & select Conversion---X or Slide Show---R.
4)Set the save directory to save the selected files.

Susie Plugin(SPI):
Grapholic can support SPI,
Go to Option-->Plugin in order to enable the selected SPI.
Restart the Grapholic to make those SPI work.
If you do not have Susie but have SPI, copy those SPI
into the Grapholic installed directory. 

Remind:
1)Because one company may have many graphic types
  or different companies use same graphic type, 
  you must find the correct one by try & error. 
  Always use the file extension or company name is enough.
  
2)There is a difference in conversion between Grapolic & Susie.
  Grapholic open all files in a archieve before carrying out conversion.
  You can find those files in the TEMPXXXX folder under the %Windows%TEMP% 
  where XXXX is a number. 
  The TEMPXXXX folder is deleted after normal termination of Grapholic,
  otherwise you must delete it by hand.

3)Also the output BMP files may have difference in color depth due to the
  different handling method between Grapholic & Susie. 
  The obvious one is that Grapholic outputs 32bit BMP but
  Susie outputs 24bit BMP.32bit BMP has alpha blending data.

4)The author of Grapholic,Wahaha Ohgoe stopped to renew Grapholic after 2000, 
  so it may not support those new games after 2000.
  Try to use SPI. There is no program to support 100% of games in the world. 

5)Some translation may be wrong due to my low standard of japanese/english,
  please feel free to contact me at zyxinnai@yahoo.com.hk to correct the faults.
