KaguyaCut      Made in HK 2009/1

Hello! KaguyaCut is a CG convertor forAtelier Kaguya games.

How to use: Basic steps are 1-4.
  1. It must have games archieve files.
  2. Open: To open *.arc or *.ari files.
  3. Folder: To select the output folder.
  4. BMP: To convert those selected items into BMPs.
  5. Cut: To rip selected items into individual files.
  6. Slider: To adjust the slide show time interval.
  7. Shortcut: Select all--Ctrl+A.
  8. Single click the preview to view it in actual size. Press Esc or Double Click to escape.
  9. Merge-->Buffer: Merge those selected items into 1 image, especially for character §W°B§§°B§U images.
  10. FacePattern: To store a face image.
  11. Char+Face: To merge one selected character images with stored face pattern.
  12. Char+Face(Auto Save): Same as above except for more than 1 character image can be selected & saving those output bmps.
  13. AlphaBlend(Buffer): To merge "right click" selected item with recent image in the preview.
  14. AlphaBlend(AutoSaver): Same as above with save.
  15. Sort Phrase: To sort those files of input extension at the end.
  16. Add LZSS Decode: Try to use LZSS method to decode those files of new extension added. Default has included extensions .cg_, .cgw, .sp_, .bg_, .parts & .dan.ÅB
  17. Load XOR Table: Input XOR table file.
  18. PlayOgg/StopOgg: To play & stop ogg files if the selected items are ogg files respectively.
  19. Play ALL ANIME FRAMES: To play anime if checked.
  20. PNG Checkbox: To save CG as PNG files if checked.
  21. F5, F11 & F12, NUMPAD 1 & 2: Refresh, Change list arrangement & Name Sort or Extension Sort.
Remark:
  1. Non support to BMR type BMP pictures.
  2. OS must support to read/write Japanese due to Atelier-Kaguya files have many Japanese filenames.
  3. 800x600 is the smallest default resolution in the merge function.
  4. Suggest to use BMPMixer to handle those characters images.
  5. The author does not afford any type of losses to users caused by using the soft.
  6. Japanese messages may be used in KaguyaCut.
  7. bass.dll is a DLL for playing ogg files.
  8. Don't delete or rename or modify files SPXOR/ONXOR/ENXOR.
  9. For School Project's & Anejiru's large characters CG: First move §§ image to the position above §W°B§U images, then select those 3 images & press 'Merge'.
