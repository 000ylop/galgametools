■ 利用のコツ --------------------------------------------------------

　これらの画像を反転や左右・上下反転などをして使うこともできます。
　たいていのルール画像は vague=1 で OK です。
　細かい模様を持ったルール画像は vague=20 ぐらいにするときれいに表示さ
れる場合があります。


■ 注意 --------------------------------------------------------------

ルール画像のファイル名には 2 バイト系文字 ( いわゆる全角文字 ) が使わ
れています。
一応問題はないはずですが、できれば、実際に使うときは 半角のみのファイル
名に改名してください。
