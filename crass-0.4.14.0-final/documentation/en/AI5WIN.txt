【Notice】
·when extracting .arc archive must specify -u parameter
·it's able to specify game parameter to extract specified game resource:
take game "Dragon Knight 4" for exsample:
crage -d g: -u ai5win -O game=DK4

game list:
Dragon Knight 4: game=DK4
らいむいろ流奇譚X cross ～恋、教ヘテクダサイ。～: game=LIME2

·if game parameter isn't specified or wanted extracting game isn't in above lsit, it will process automatically prediction(not 100% successful) to extract target resource. if it occurs extraction error, please visit https://www.yukict.com/bbs/forumdisplay.php?fid=69&page=1 to report error(mark game name and upload to minimal.arc file)
