【.dat/.dmf封包文件说明】
    实际类型是有目录的封包资源文件。
    元数据根据实际用途，是png、ogg、scn等（还可以解出资料这个目录，内有脚本语法和打包程序）。
    
    	首部：
    		s8 magic[4];	// "DMF "
		u32 entries;	// 资源文件数（每项长度变长）
	索引段：
		s8 *name;	// 资源文件名
		u32 name_length;// 资源文件名长度
		u32 offset;	// 资源文件数据偏移（从文件头算起）	
		u32 length;	// 资源文件数据长度	
	数据：
		紧接在索引段之后。
