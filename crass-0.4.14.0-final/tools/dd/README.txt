文件分割器dd

其用途是当使用crass提取失败时，为了给开发者提供有用的文件，通常用该工具把目标文件（通常很大）切分为小块。
请在控制台中运行dd查看具体使用方法。

这里简要介绍一下。

Usage: dd -i [input file] [-o [output dir]] [-p [position]] -s [block size] [-c
[blocks]]

  -i  指定要分割的文件
  -o  指定输出的块文件保存的路径 (默认: 当前目录下的dd_output_dir目录中)
  -p  从输入文件的哪个位置开始分块起始位置. (默认: 0)
  -s  每个块文件的大小.
  -c  分多少块. (默认: 整个文件被分割的最大块数)

举例：
  dd -i foo.file -s 1000
  => 将文件foo.file分成1000字节大小的块文件。

  dd -i foo.file -o foo.blocks -p 0x100 -s 0x1000 -c 3
  => 将文件foo.file分成3块，每块4096(用16进制表示为0x1000)字节，分块的起始位置从输入文件256(用16进制表示为0x100)字节处开始。
