kirikiri2 には吉里吉里２
kag3 にはKAG3

が入っています。

license.txt には吉里吉里/KAGのライセンスが書いてあります。
良くお読みになった上でこのソフトウェアを使用してください。


kirikiri2 および kag3 のフォルダの readme.txt には、配布ファイルの
概要やリリースノートが記述されています。とくにリリースノートは、
前バージョンからの変更点やバージョンアップに伴う注意点などが書かれ
ているので、お読みください。


紹介用スクリプトは kag3/syoukai にあります。
これを実行するには、kirikiri2 フォルダにある krkr.eXe を実行し、
フォルダ/アーカイブの選択 のダイアログボックスで kag3/syoukai を
選択してください。


Warning:
This software includes JPEG decoder/encoder. Forgent Networks is
claiming a patent used in JPEG compression. Do not use this software
until you verify the patent-related issue (Japanese users do not need
to worry about it because the patent is not valid in Japan).
