【概    要】
  このプログラムは BMP 形式のデータを抽出する Media Extractor のプラグインです。
  日本語版 Windows Vista / XP、WESISIDE Media Extractor が正常に動作するコンピュ
  ータで実行可能です。

【対応フォーマット】
  Bitmap(BMP) 形式

【導入方法】
  WESISIDE Media Extractor の plugins ディレクトリ内に、このプラグインファイル
  「ExtractBMP.dll」を格納してください。

【操作方法】
  WESISIDE Media Extractor の「プラグイン設定」画面内に、本プラグインが表示され
  ますので、必要に応じて有効・無効の設定を行なってください。
  操作方法については、WESISIDE Media Extractor のマニュアルをご参照下さい。

【詳細設定】
  このプラグインは、抽出したデータを簡易編集することができます。
  WESISIDE Media Extractor の「プラグイン設定」画面内に表示された本プラグインを
  選択すると、「プロパティ」ボタンが有効になりますので、これをクリックすると詳細
  設定画面が開きます。

【詳細設定画面操作方法】
  「色変換」グループ
      抽出データのカラー情報を色素単位で入れ替えます。
      抽出したデータの色調が不自然な場合に設定してください。
      抽出したカラー情報が RGB の順で格納されているものとした上で、どのように色
      素を入れ替えるか設定します。
      ※色変換は抽出したデータが、8bit / 24bit カラーの時のみ動作します。
  「画像反転」グループ
      抽出データの画素を反転します。
      抽出したデータにおける画像の上下が逆の場合に設定してください。

【制限事項】
    ※抽出結果の最大ファイルサイズは 4GBytes です。
    ※データ部が暗号化されているなどの理由により、抽出結果のファイルが正しい
    　フォーマットでない場合があります。 
    ※本ツールは，Windows Vista / XP 用です。
    　その他のWindowsでの動作確認はしていません。
    ※変換時、ドライブの空き容量の計算を行っておりませんので
    　変換ファイルを出力するドライブには十分な空きを作っておいて下さい。
