--------------------------
d3dslide 1.1 (c) asmodean
--------------------------

d3dslide is an image viewer optimized to display images of any aspect-ratio
fullscreen with pleasing animation effects.

See d3dslide.ini for various adjustable options.

-----
USAGE
-----

Run the program and select a directory with images.

Actions:

- Next image: 
    Left mouse button
    Right arrow key
    Page down key

- Previous image:
    Right mouse button
    Left arrow key
    Page up key

- Pause/Unpause:
    Spacebar key

- Exit:  
    Escape key

-------------------
SYSTEM REQUIREMENTS
-------------------

- Windows XP, Vista x32/x64, Windows 7 x32/x64 etc
- DirectX 9.0c
- GPU with Vertex Shader 1.1 and Pixel Shader 2.0 support
